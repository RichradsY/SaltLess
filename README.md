# SaltLess

SaltLess is a wellness tracking app that helps users manage daily salt intake with simple and clear logging.

## Overview

With SaltLess, you can easily understand:

- How much salt you consumed today
- How much salt you intentionally reduced
- Whether your long-term habits are improving

## Key Features

- Two logging modes
  - Tracking Mode: log actual intake
  - Zen Mode: log intentional reduction
- Sodium/salt conversion assistance
- Daily and historical summaries
- Multilingual support (including Chinese and English)
